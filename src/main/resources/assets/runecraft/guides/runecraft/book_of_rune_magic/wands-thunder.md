# Thunder Wand

<ItemImage id="runecraft:thunder_wand" />

**__Effects__** 

**Right Click:** 
Changes the weather to sunshine. 

**Right Click + Shift:** 
Changes the weather to rain. 

**Left Click:** 
Sends a lightning bolt to the creature.
